张国华，教授，博士生导师，现工作于北京科技大学钢铁冶金新技术国家重点实验室。本课题组隶属于周国治院士团队，主要研究方向为，i) 冶金熔体物理化学；ii) 固废综合利用；iii) 难熔金属冶金，包括钨钼冶金，铁合金制备及难熔金属碳化物、氮化物、硼化物、硅化物制备。

2006年以总成绩年级第一毕业于北京科技大学冶金工程专业，并保送本校研究生。2012年取得北京科技大学冶金工程专业博士学位，导师为周国治院士；分别于2013.6-2014.4和2017.9-2018.3在日本东京大学(合作导师：Prof. Tsukihashi Fumitaka)和美国犹他大学(合作导师：Prof. Zhigang Zak Fang)做访问学者。2014年被聘为副教授，2017年被聘为博士生导师，2018年被聘为教授。迄今为止，共在国内外公开学术刊物发表论文160余篇，其中以第一/通讯作者身份发表SCI检索论文130余篇，获授权国家发明专利10余项。入选中国科协第一届“青年人才托举工程”项目(2015年)；获评“北京市优秀博士学位论文”(2013年)；第七届中国青少年科技创新奖(2011年)。
